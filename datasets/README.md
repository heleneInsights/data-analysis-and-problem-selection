# Datasets

This project analyzes four datasets obtained from public and trusted sources.

| Dataset | Source |
|----------|---------|
| Aircraft Registrations in Chile | Chilean Government Open Data Portal – Aircraft registrations in the National Aircraft Registry (R.N.A.) |
| Earthquakes in Chile | Kaggle – Earthquakes in Chile Dataset |
| Healthy Diet and Calorie Intake | Kaggle – Healthy Diet and Calorie Intake Dataset |
| Daily Temperatures in Chile | Chilean Government Open Data Portal – Historical Temperature Records |

## Sources

- Aircraft Registrations in Chile: https://datos.gob.cl/dataset/aeronaves-inscritas-en-el-r-n-a-al-30-abr-2026-xlsx
- Earthquakes in Chile: https://www.kaggle.com/datasets/nicolasgonzalezmunoz/earthquakes-on-chile
- Healthy Diet and Calorie Intake: https://www.kaggle.com/datasets/aliyasaly1231/healthy-diet-and-calorie-intake
- Daily Temperatures in Chile: https://datos.gob.cl/dataset/2725

## Dataset Selection Criteria

The datasets were selected to provide diversity in domains, data structures, and potential machine learning applications, including classification, regression, clustering, and forecasting tasks.
